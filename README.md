# responsive-site2.0
El taller #1 Layout Responsive se incluyeron los primeros conocimientos obtenidos en cuanto estructuras html y css. Dentro del archivo index.html se utilizaron elementos html, header, nav, body, footer. Ademas se incluyeron items como href# para incluir links dentro del Wireframe, p (parrafos), img (imagenes) y tablas. 

En cuanto al archivo Style.css se tuvo en cuenta principlamente la metodologia Mobile first, esto para que el responsive tenga la capacidad de ser abierto en diferentes tipos de pantalla como mobile, tablet y desktop, por medio de @media. Alli, ademas se modifico el tipo de letra de cada uno de los items como titulos, subtitulos y tablas, tambien se modifico la calidad de las imagenes, el color base de nuestro Wireframe y de cada aspecto que incluye.
